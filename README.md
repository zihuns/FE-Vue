# FE-Vue

이 저장소는 Vue.js 프론트엔드 개발 기술을 학습하고 실습한 코드들을 모아둔 공간입니다.

## 📂 프로젝트 목록 (Projects)

### 1. [vue3-test](./vue3-test)

Webpack을 사용하여 수동으로 구성된 Vue 3 기본 예제 프로젝트입니다.

- **주요 내용**: Vue 3 기본 컴포넌트 시스템, 반응형 데이터, Webpack 설정
- **기능**: 카운터, 과일 목록 렌더링

### 2. [vue3-template-syntax](./vue3-template-syntax)

Vue 3의 템플릿 문법(Template Syntax)을 학습하기 위한 프로젝트입니다.

- **주요 내용**: 데이터 바인딩(`v-bind`, `v-model`), 이벤트 처리, 조건부/리스트 렌더링
- **특징**: Webpack 5 기반 개발 환경 템플릿

### 3. [vue3-movie-app](./vue3-movie-app)

OMDb API를 활용하여 영화 검색 및 상세 정보를 조회하는 애플리케이션입니다.

- **기술 스택**: Vue 3, Vue Router, Vuex, Axios, Bootstrap 5
- **기능**: 영화 검색, 상세 정보 조회, 라우팅 시스템
  Vue 3의 핵심 개념부터 Webpack을 활용한 개발 환경 구성까지 다양한 예제를 포함하고 있습니다.
